# SpringBootStudy

Projeto CRUD de um estacionamento implementado para revisão de conceitos básicos de API Rest utilizzando Java com Spring Boot
Implemtação de uma API Rest - Curso básico desenvolvido pela profª Michelli Brito link:(https://www.youtube.com/watch?v=LXRU-Z36GEU)

Tecnologias utilizaas:
JDK 11
Spring Boot
Spring MVC
Spring Data JPA
Spring Validation
Banco de dados: PgAdmin (PostgreSQL)
Consultas API: PostMan
IDE STS
